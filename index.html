Начать с Урока 0

Этот курс создан для ученика без базовой подготовки. Мы приучаем работать с документацией на php.net с первого дня и двигаемся от простого к необходимому для реальной разработки.

Формат каждого занятия: мини‑теория, живые примеры, 2–3 задачи и ссылка на первоисточник. Домашка — маленький практический блок, который можно проверить прямо в браузере или консоли.

Урок 0. Подготовка окружения

Цель: установить и запустить PHP, настроить рабочую папку и запустить первый скрипт через браузер и консоль.

Что такое PHP

PHP — это интерпретируемый язык, исполняется на стороне сервера. Страница index.php сначала выполняется PHP‑интерпретатором, а затем её результат отдается браузеру.

Установка PHP

Windows

  1. Зайдите на windows.php.net/download
  2. Скачайте Thread Safe ZIP последней стабильной версии
  3. Распакуйте в C:\php и добавьте этот путь в переменную PATH
  4. Проверьте в терминале: php -v

Альтернатива: XAMPP (Apache+PHP+MySQL).

macOS

  1. Установите Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
brew install php
php -v

Linux (Ubuntu/Debian)

sudo apt update
sudo apt install php
php -v

Рабочая папка и первый файл

C:\php-projects
└─ lesson0

Создайте index.php со строками:

<?php
echo "Привет, мир!";

Запуск

  1. Встроенный сервер: php -S localhost:8000 → откройте http://localhost:8000
  2. Консольный запуск: php index.php

Домашка

  • Поставить PHP и запустить встроенный сервер
  • Создать hello.php, вывести имя и город
  • Запустить файл и в браузере, и через консоль

Урок 1. Первое знакомство с PHP

Тезисы: теги <?php ?>, вывод echo/print/var_dump, базовые ошибки, цикл «правка‑перезапуск‑проверка».

Мини‑задачи

  1. Выведите имя и возраст одной строкой.
  2. Сложите два числа и выведите результат.

Документация: echo, var_dump.
